Unblockable Video API

Canvas video streaming without video, audio, or media tags.

Back to Player

Render Hosting

Build Command: pip install -r requirements.txt
Start Command: python server.py --host 0.0.0.0 --port $PORT

FFmpeg must be available on the server for audio and best video performance.

HTTP API

GET / Player page.
GET /health Health check for Render.
GET /api_doc.html API documentation page.
GET /api/status?src=... Returns video metadata and current server settings.
POST /api/source Sets the default source. JSON body: {"src":"/media/input.mp4"}
POST /api/upload Uploads a video. Multipart field name: file. Returns src and metadata.
GET /api/uploads Lists uploaded videos in the media folder.
DELETE /api/upload/{name} Deletes an uploaded video.
GET /api/player.js Embed helper script for custom source tags.
GET /api/embed?src=... Returns ready-to-copy embed HTML.

WebSocket API

/api/frames Binary JPEG frame stream. Query: src, start, speed, width, fps, quality.
/api/audio Binary 48 kHz stereo PCM stream. Query: src, start, speed.

Upload Example

curl -F "file=@video.mp4" https://your-render-app.onrender.com/api/upload

Source Example

curl -X POST https://your-render-app.onrender.com/api/source \
  -H "content-type: application/json" \
  -d "{\"src\":\"/media/input.mp4\"}"

Embed Example

<src width="960px" height="540px">https://example.com/video.mp4</src>
<script src="https://your-render-app.onrender.com/api/player.js"></script>

Alternative Embed

<unblockable-video src="https://example.com/video.mp4" width="960px" height="540px"></unblockable-video>
<script src="https://your-render-app.onrender.com/api/player.js"></script>
