Unblockable Video API

Canvas video streaming without video, audio, or media tags.

Render Hosting

Build Command: pip install -r requirements.txt
Start Command: python server.py --host 0.0.0.0 --port $PORT

FFmpeg must be available on the server for audio and best video performance.

Embed

Place this on another page. The helper script replaces the custom source tag with the player iframe.

<src width="960px" height="540px">https://example.com/video.mp4</src>
<script src="https://your-render-app.onrender.com/api/player.js"></script>

This also works:

<unblockable-video src="https://example.com/video.mp4" width="960px" height="540px"></unblockable-video>
<script src="https://your-render-app.onrender.com/api/player.js"></script>

HTTP API

GET /Player page.
GET /healthHealth check for Render.
GET /api_doc.htmlThis documentation.
GET /api/status?src=...Returns metadata and current server settings.
POST /api/sourceSets the default source. JSON body: {"src":"/media/input.mp4"}
POST /api/uploadUploads a video. Multipart field name: file. Returns src and metadata.
GET /api/uploadsLists uploaded videos in the media folder.
DELETE /api/upload/{name}Deletes an uploaded video.
GET /api/player.jsEmbed helper script for custom source tags.
GET /api/embed?src=...Returns ready-to-copy embed HTML.

WebSocket API

/api/framesBinary JPEG frame stream. Query: src, start, speed, width, fps, quality.
/api/audioBinary 48 kHz stereo PCM stream. Query: src, start, speed.

Upload Example

curl -F "file=@video.mp4" https://your-render-app.onrender.com/api/upload
